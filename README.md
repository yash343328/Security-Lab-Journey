# Security Lab Journey

This repository documents my learning journey in cybersecurity.  
It is maintained as a daily security lab journal.

The aim of this repository is to understand how computer systems are attacked, where security weaknesses exist, and how those weaknesses can be defended.

* * *

## What This Repository Contains

This is not a tool-based or copy-paste notes repository.  
Each topic is written with a security-thinking approach.

For every concept, the focus is on:

-   Attacker point of view
    
-   Vulnerability or weak configuration
    
-   Defense and prevention methods
    
-   Relation to real-world security incidents
    

* * *

## Learning Method

Topics are explored step by step using practical reasoning.

The learning process focuses on:

-   Understanding why an attack works
    
-   Identifying trust and security failures
    
-   Studying how defenders detect and respond
    
-   Connecting theory with real-world cases
    

* * *

## Topics Covered

-   Cybersecurity basics
    
-   Network and system security
    
-   Linux security fundamentals
    
-   Web application security concepts
    
-   Common attack techniques
    
-   Blue team defense concepts
    
-   Red team attack perspectives
    
-   Introduction to AI in cybersecurity
    
-   Analysis of real-world breaches
